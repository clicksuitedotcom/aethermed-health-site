# AetherMed Health Website

Static website for AetherMed Health, an international medical service helping patients coordinate care in China.

## Local Preview

Open `index.html` directly in a browser, or serve this folder with any static file server.

## Deployment

Recommended production flow:

1. Push this repository to GitHub.
2. Connect the GitHub repository to Cloudflare Pages.
3. Use the project root as the output directory.
4. No build command is required.
5. Add `aethermed.health` and `www.aethermed.health` as custom domains in Cloudflare Pages.
6. Update DNS records in GoDaddy as instructed by Cloudflare.

## Files

- `index.html`: site content and structure
- `styles.css`: visual design and responsive layout
- `script.js`: language switcher and demo form interaction
